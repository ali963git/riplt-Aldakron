📖 نبذة عن المشروع 

الذَّاكِرُون (Aldakron) هو مشروع تقني إسلامي يهدف إلى تقديم تجربة رقمية متكاملة تساعد المسلمين في الوصول إلى:

القرآن الكريم بطريقة سهلة وحديثة. التلاوات الصوتية لعدد من القراء. الأذكار اليومية. مواقيت الصلاة حسب الموقع. اتجاه القبلة باستخدام البوصلة. أدوات إسلامية ذكية مدعومة بالذكاء الاصطناعي. 

تم تصميم المنصة لتكون سريعة، متجاوبة، وقابلة للتوسع لتصبح تطبيقاً عالمياً يعمل على الويب والجوال.

✨ المميزات 📖 القرآن الكريم عرض القرآن الكريم كاملاً (114 سورة). دعم النص القرآني بالتشكيل. بحث سريع داخل الآيات. حفظ آخر قراءة. العلامات المرجعية. التحكم بحجم الخط. الوضع الليلي للقراءة. تشغيل التلاوات الصوتية. 🎧 التلاوات مكتبة قراء متعددة. تشغيل مباشر للتلاوات. حفظ السورة المفضلة. دعم التشغيل المستمر. 🤲 الأذكار
