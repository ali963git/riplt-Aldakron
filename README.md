## ✨ نبذة عن المشروع

**الذَّاكِرُون (Aldakron)** هو مشروع تقني إسلامي يجمع بين المحتوى الديني والتقنيات الحديثة لإنشاء منصة متكاملة تساعد المستخدم على:

- 📖 قراءة القرآن الكريم
- 🎧 الاستماع للتلاوات
- 🤲 متابعة الأذكار اليومية
- 🕋 معرفة اتجاه القبلة
- 🕌 متابعة مواقيت الصلاة
- 🤖 استخدام مساعد ذكي للمعلومات الإسلامية

---

## 🚀 المميزات

### القرآن الكريم
- عرض السور كاملة (114 سورة)
- دعم التشكيل العربي
- البحث في الآيات
- الاستماع لعدة قراء

### الأذكار
- أذكار الصباح والمساء
- عداد التسبيح
- تذكيرات يومية

### الصلاة
- حساب مواقيت الصلاة حسب الموقع
- تنبيهات الصلاة
- اتجاه القبلة باستخدام البوصلة

### الذكاء الاصطناعي
- مساعد إسلامي ذكي
- البحث في المحتوى الإسلامي
- إجابات منظمة مع مراعاة المصادر
